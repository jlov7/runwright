# Getting Started

Pick the path that matches your role and goal:

- Build and run now (technical): `docs/getting-started/quickstart.md`
- Understand value and governance (non-technical): `docs/getting-started/non-technical-onboarding.md`
- See end-to-end persona flows: `docs/getting-started/user-journeys.md`

If you are unsure what to do in your current project state, run:

```bash
pnpm tsx src/cli.ts journey
```

If you are blocked, go to `docs/help/troubleshooting.md`.
