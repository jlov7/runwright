# Frontend Governance Cadence

## Weekly quality review
- Review score deltas across 12 frontend criteria.
- Validate gate artifacts: a11y, visual, perf, coverage, journey E2E.
- Track open/high findings and assign owners.

## Escalation workflow
- Design blockers: escalate to design owner within 24h.
- Engineering blockers: escalate to runtime/frontend owner within 24h.
- Product blockers: escalate to release lead for scope/risk decision.
