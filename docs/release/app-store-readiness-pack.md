# App-store Readiness Pack

## Metadata
- Product name, subtitle, and category mapping.
- Short and long descriptions aligned to actual feature set.
- Keywords and localization variants.

## Creative Assets
- Required icon sizes.
- Screenshot set for onboarding, gameplay, social, and creator flows.
- Optional trailer/highlight reel with gameplay timeline.

## Policy Disclosures
- Privacy policy URL.
- Data collection summary (telemetry, moderation, crash diagnostics).
- Content moderation and UGC reporting policy.

## Submission Checklist
- Build signed and verified.
- Release notes prepared.
- Age rating questionnaire completed.
- Support URL and contact verified.
- Rollback runbook linked in release ticket.
