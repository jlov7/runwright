# Frontend Pre-release Operations Pack

## Dry-run checklist (owner signoff)
- Product: journey/copy/help readiness signoff.
- Design: visual consistency and responsive/a11y signoff.
- Engineering: tests/coverage/perf/CI signoff.
- Release: final gate artifact verification signoff.

## Panel demo script
1. Launch dashboard and show wayfinding plus command-bar navigation.
2. Run onboarding first-success flow (profile -> tutorial -> save -> publish).
3. Trigger ranked failure and show clear nextAction recovery guidance.
4. Demonstrate retry queue and diagnostics packet copy.
5. Show creator/liveops/analytics/help surfaces and readiness state.

## Post-demo feedback intake
- Capture findings by severity (`critical`, `major`, `minor`).
- Assign owners and deadlines within one business day.
- Prioritize high-impact UX and reliability fixes into the next fast-fix sprint.
